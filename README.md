# Cricket World Cup Insights (2019)

This project is a Business Intelligence (BI) dashboard created using Google Looker Studio to analyze Cricket World Cup 2019 match data.

## Objective
To visualize historical cricket data and identify patterns related to team performance, trends, and venue-based insights.

## Tools Used
- Google Looker Studio
- Google Sheets
- Kaggle Dataset (Cricket World Cup)

## Key Features
- Team-wise performance analysis using total runs
- Interactive filters for team comparison
- Trend analysis of highest match scores
- Venue-wise performance analysis
- Match-level analytics table
